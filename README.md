# UC4 - Banner App (Array & Loops)

##  Description

This project is a Java program that prints a stylized banner using **String arrays** and **loops**.
It demonstrates how to avoid repetitive print statements by storing banner lines in an array and printing them using an enhanced for-loop.

---

##  Objective

* Use **String array** to store banner lines
* Use **looping (for-each loop)** to print output
* Improve **code reusability and modularity**

---

##  Technologies Used

* Java
* Arrays
* Enhanced For Loop
